# こども園 財務みえる化

WAM NET「社会福祉法人の財務諸表等電子開示システム」の公開情報から、こども園等の財務状況をやさしく確認するWebアプリです。

## まず起動する

Windowsでは、このフォルダにある **「起動する.bat」** をダブルクリックしてください。

数秒待つとブラウザで `http://127.0.0.1:5000` が開きます。黒い画面はアプリ本体なので、使っている間は閉じないでください。

うまく起動しない場合は、このフォルダに作られる `startup.log` の内容を確認してください。

## 使い方

1. 都道府県を選ぶ
2. 市町村を選ぶ
3. 表示された候補から園名を選ぶ
4. 「この園の財務状況を見る」を押す

候補が多い場合は「園名で絞り込み」に園名の一部を入れて「候補を更新」を押してください。

## できること

- 都道府県 → 市町村 → 園名の選択式検索
- WAM NETから最新の計算書類ZIPを取得
- 事業活動計算書・貸借対照表・資金収支計算書PDFの自動読取
- 園・拠点区分を特定できる場合はその区分、できない場合は運営法人全体を表示
- 5段階のやさしい財務診断
- 一般的な改善戦略を2つ表示
- Instagram、メール、問い合わせフォームへの導線
- スマホ表示対応

## 大切な注意

- 本アプリは公開資料にもとづく簡易確認用です。会計・投資・就職判断の専門的助言ではありません。
- 園単独の資料が公開PDF内で特定できない場合、運営法人全体の結果を表示します。
- WAM NET側の画面や帳票様式が変わると、検索やPDF読取の調整が必要になる場合があります。
- 利用時はWAM NETの利用条件や出典表示に従ってください。
